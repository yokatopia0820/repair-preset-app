# 修理報告プリセット支援ツール

修理報告フォームの定型入力をプリセットとして保存し、必要な項目へ一括反映するためのWebアプリです。

## 主な機能

- 主要事項、製品情報、作業報告、交換部品、請求/支払の入力タブ
- プリセットの保存、読込、上書き、削除
- JSONでのバックアップ書き出し、読み込み
- 一時保存
- 入力サマリコピー、タブ単位コピー、項目コピー
- Firebase Firestoreを使った端末間プリセット同期

## GitHub Pagesで公開する場合

このリポジトリをGitHub Pagesで公開すると、スマホ、iPad、PCから同じURLでアクセスできます。

GitHubのリポジトリ設定で `Settings > Pages` を開き、公開元を `main` ブランチの `/root` に設定してください。

## 端末間同期について

GitHub Pagesは静的サイト公開だけなので、保存データを端末間で共有するにはクラウドDBが必要です。

このアプリでは Firebase Firestore のWeb設定JSONをアプリ内の「プリセット管理 > クラウド同期」に貼り付けることで、同じ同期IDを使う端末間でプリセットを送受信できます。

詳しい設定手順は [firebase-setup.md](firebase-setup.md) を参照してください。

## ローカルで確認する場合

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

または、同梱の `server.js` をNode.jsで起動できます。
