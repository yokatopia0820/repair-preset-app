# 修理報告プリセット支援ツール

修理報告フォームの定型入力をプリセットとして保存し、必要な項目へ一括反映するためのWebアプリです。

## 主な機能

- 主要事項、製品情報、作業報告、交換部品、請求/支払の入力タブ
- プリセットの保存、読込、上書き、削除
- 一時保存
- 項目コピー
- 編集モード / 表示モードの切り替え
- 表示モードで入力済み項目をタップしてコピー
- iPhone幅でも5つのタブが画面内に収まるレスポンシブ表示

## GitHub Pagesで公開する場合

このリポジトリをGitHub Pagesで公開すると、スマホ、iPad、PCから同じURLでアクセスできます。

GitHubのリポジトリ設定で `Settings > Pages` を開き、公開元を `main` ブランチの `/root` に設定してください。

## 保存データについて

保存したプリセットは、使っている端末のブラウザ内に保存されます。

スマホ、iPad、PCの間で自動同期するには、別途クラウド保存用のサーバーやデータベースが必要です。現在の版では、現場で迷わず使えるように、保存と読み込みを中心にしたシンプルな構成にしています。

クラウド同期を組み込む場合も、画面上は保存と読み込みだけに保ち、裏側の保存先だけクラウドDBへ切り替える想定です。

## ローカルで確認する場合

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

または、同梱の `server.js` をNode.jsで起動できます。
